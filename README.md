### 💰 FIZ — Finanzas personales

App full-stack para registrar movimientos, administrar cuentas multi-moneda y visualizar métricas mensuales por categoría. Auth con JWT, balances atómicos y despliegue con Docker.

**Frontend:** Angular 21 (signals, standalone) · PrimeNG · Tailwind
**Backend:** Express 5 · PostgreSQL · Sequelize · JWT
**Infra:** Docker · GitHub Actions

[Frontend →](https://github.com/freykill/fiz-ng) · [Backend →](https://github.com/freykill/fiz-back)
